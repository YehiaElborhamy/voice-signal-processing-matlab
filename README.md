# Voice Signal Processing Using MATLAB

A signal processing project focused on analyzing and modifying voice signals in both time and frequency domains using MATLAB.

---

# 🚀 Project Overview

This project demonstrates several digital signal processing techniques applied to voice signals using MATLAB.

The project includes:

* Time-domain signal processing
* Frequency-domain analysis using FFT
* Signal manipulation and audio effects
* Audio visualization and playback

---

# ⚡ Features

* Voice signal recording and processing
* Signal reversal
* Tone addition
* Echo generation
* Frequency spectrum analysis using FFT
* WAV audio file generation
* Time-domain and frequency-domain visualization

---

# 🛠 Technologies Used

* MATLAB
* Digital Signal Processing (DSP)
* FFT (Fast Fourier Transform)
* Audio Signal Analysis

---

# 🔬 Processing Techniques Used

## Time Domain Processing

* Signal reversal
* Echo effect generation
* Tone addition

## Frequency Domain Processing

* Fast Fourier Transform (FFT)
* Frequency spectrum visualization
* Signal frequency analysis

---

# 📊 MATLAB Operations Applied

* `audioread()`
* `audiowrite()`
* `plot()`
* `fft()`
* Signal manipulation and filtering

---

# 🎵 Audio Processing Workflow

1. Read the original voice signal
2. Visualize the signal in time domain
3. Reverse the signal
4. Add generated tone
5. Generate echo effect
6. Analyze frequency spectrum using FFT
7. Save processed audio signals
